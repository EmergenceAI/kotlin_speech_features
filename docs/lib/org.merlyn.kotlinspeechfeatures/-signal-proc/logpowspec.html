fun logpowspec(    frames: Array<FloatArray>,     nfft: Int,     norm: Boolean = true): Array<FloatArray>

Compute the log power spectrum of each frame in frames. If frames is an NxD matrix, output will be Nx(NFFT/2+1).

Return

If frames is an NxD matrix, output will be Nx(NFFT/2+1). Each row will be the log power spectrum of the corresponding frame.

Parameters

frames

the array of frames. Each row is a frame.

nfft

the FFT length to use. If NFFT frame_len, the frames are zero-padded.

norm

If norm=true, the log power spectrum is normalised so that the max value (across all frames) is 0.
